## 🎯 Objetivo

Refatorar o código HTML original visando:

- Aplicação correta de semântica HTML5
- Separação entre estrutura (HTML) e apresentação (CSS)
- Melhoria da acessibilidade
- Padronização e organização do código
- Conformidade com boas práticas de desenvolvimento front-end

---

## 🔎 Problemas Identificados

1. Uso excessivo de `<div>` sem significado semântico, comprometendo SEO e acessibilidade.
2. CSS embutido no HTML, dificultando manutenção e reutilização.
3. Estrutura incorreta de títulos (uso de div no lugar de h2 e h3).
4. Imagem sem atributo `alt`, prejudicando acessibilidade.
5. Menu de navegação sem estrutura de lista (`ul` e `li`), comprometendo semântica.
